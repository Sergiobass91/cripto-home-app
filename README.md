# CryptoHome
Cotización en tiempo real de criptomonedas en diferentes divisas, seguimiento de precios, login de usuarios para tener stack de portafolio.
Este repositorio es usado para Práctica profesional 3 en el IFTS N° 18

# Tecnologías
React + React-Router + Vite + TailwindCSS, REST API...en proceso

# Arquitectura limpia
La meta es lograr una aplicación profesional, con independencia de servicios, escalable y segura.
